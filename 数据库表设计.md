# 数据库表结构设计

---

## 表 4-1 用户信息表 (sys_user)

| 序号 | 字段名称 | 字段类型 | 字段说明 | 约束 |
|:----:|:---------|:---------|:---------|:-----|
| 1 | id | BIGINT | 主键，自增 | PK |
| 2 | username | VARCHAR(50) | 用户名/账号 | NOT NULL, UNIQUE |
| 3 | password | VARCHAR(255) | 加密后的密码 | NOT NULL |
| 4 | role | INT | 用户角色标识<br>1-用户 2-店铺 3-技师 4-管理员 | NOT NULL |
| 5 | phone | VARCHAR(20) | 联系电话 | |
| 6 | real_name | VARCHAR(100) | 真实姓名 | |
| 7 | email | VARCHAR(100) | 电子邮箱 | |
| 8 | avatar | VARCHAR(500) | 头像URL | |
| 9 | skill | VARCHAR(500) | 专业技能（技师用） | |
| 10 | status | INT | 账号状态 0-禁用 1-正常 | DEFAULT 1 |
| 11 | verified | INT | 是否认证 0-未认证 1-已认证 | DEFAULT 0 |
| 12 | create_time | DATETIME | 创建时间 | NOT NULL |
| 13 | update_time | DATETIME | 更新时间 | NOT NULL |
| 14 | deleted | INT | 逻辑删除 0-已删除 1-正常 | DEFAULT 0 |

## 表 4-2 维修店信息表 (repair_shop)

| 序号 | 字段名称 | 字段类型 | 字段说明 | 约束 |
|:----:|:---------|:---------|:---------|:-----|
| 1 | id | BIGINT | 主键，自增 | PK |
| 2 | user_id | BIGINT | 店主ID，关联 user 表 | FK, NOT NULL |
| 3 | shop_name | VARCHAR(100) | 维修店名称 | NOT NULL |
| 4 | address | VARCHAR(500) | 详细地址 | NOT NULL |
| 5 | phone | VARCHAR(20) | 联系电话 | NOT NULL |
| 6 | description | TEXT | 店铺简介 | |
| 7 | business_hours | VARCHAR(200) | 营业时间 | |
| 8 | license | VARCHAR(500) | 营业执照 | |
| 9 | cover_image | VARCHAR(500) | 店铺封面图 | |
| 10 | rating | DOUBLE | 店铺评分 | DEFAULT 5.0 |
| 11 | status | INT | 营业状态 0-歇业 1-营业 | DEFAULT 1 |
| 12 | audit_status | INT | 审核状态 0-待审核 1-通过 2-拒绝 | DEFAULT 0 |
| 13 | auto_confirm | INT | 自动确认订单 0-手动 1-自动 | DEFAULT 0 |
| 14 | create_time | DATETIME | 创建时间 | NOT NULL |
| 15 | update_time | DATETIME | 更新时间 | NOT NULL |
| 16 | deleted | INT | 逻辑删除 | DEFAULT 0 |

## 表 4-3 预约订单表 (appointment)

| 序号 | 字段名称 | 字段类型 | 字段说明 | 约束 |
|:----:|:---------|:---------|:---------|:-----|
| 1 | id | BIGINT | 预约单ID，主键 | PK |

| 5 | service_id | BIGINT | 服务类型ID | FK |
| 6 | employee_id | BIGINT | 负责技师ID | FK |
| 7 | motorcycle_brand | VARCHAR(50) | 摩托车品牌 | |
| 8 | motorcycle_model | VARCHAR(50) | 摩托车型号 | |
| 9 | problem_description | TEXT | 故障描述 | |
| 10 | appointment_time | DATETIME | 预约到店时间 | NOT NULL |
| 11 | status | INT | 订单状态<br>0-待确认 1-已确认 2-维修中 3-已完成 4-已取消 | NOT NULL |
| 12 | remark | VARCHAR(500) | 订单备注 | |
| 13 | total_amount | DECIMAL(10,2) | 订单总金额 | |
| 14 | pay_status | INT | 支付状态 0-未支付 1-已支付 | DEFAULT 0 |
| 15 | pay_method | VARCHAR(20) | 支付方式 | |
| 16 | pay_time | DATETIME | 支付时间 | |
| 17 | complete_time | DATETIME | 完成时间 | |
| 18 | tow_service | INT | 是否需要拖车 0-否 1-是 | DEFAULT 0 |
| 19 | tow_address | VARCHAR(500) | 拖车地址 | |
| 20 | create_time | DATETIME | 创建时间 | NOT NULL |
| 21 | update_time | DATETIME | 更新时间 | NOT NULL |
| 22 | deleted | INT | 逻辑删除 | DEFAULT 0 |

## 表 4-4 维修记录表 (repair_record)

| 序号 | 字段名称 | 字段类型 | 字段说明 | 约束 |
|:----:|:---------|:---------|:---------|:-----|
| 1 | id | BIGINT | 记录ID，主键 | PK |
| 2 | appointment_id | BIGINT | 关联的预约订单ID | FK, NOT NULL |
| 3 | technician_id | BIGINT | 维修技师ID | FK |
| 4 | fault_description | TEXT | 故障详细描述 | NOT NULL |
| 5 | repair_content | TEXT | 维修内容 | |
| 6 | parts_used | TEXT | 使用的配件及耗材记录 | |
| 7 | total_cost | DECIMAL(10,2) | 最终维修总费用 | NOT NULL |
| 8 | repair_start_time | DATETIME | 维修开始时间 | |
| 9 | repair_end_time | DATETIME | 维修结束时间 | |
| 10 | create_time | DATETIME | 创建时间 | NOT NULL |
| 11 | update_time | DATETIME | 更新时间 | NOT NULL |
| 12 | deleted | INT | 逻辑删除 | DEFAULT 0 |

## 表关系ER图

```
├─────────────────┤         ├─────────────────┤         ├─────────────────┤
│ PK  id         │──┐  1:N │ PK  id         │──┐  1:N │ PK  id          │
│    username    │  │       │ FK  user_id    │  │       │ FK  user_id    │
│    password    │  │       │    shop_name   │  │       │ FK  shop_id    │
│    role        │  │       │    address     │  │       │ FK  service_id │
│    phone       │──┘       │    status      │──┘       │    status      │
└─────────────────┘         └─────────────────┘         └────────┬────────┘
                                                                  │
                                                                  │ N:1
                                                                  ▼
                                                         ┌─────────────────┐
                                                         │  repair_record  │
                                                         │   (维修记录表)   │
                                                         ├─────────────────┤
                                                         │ PK  id          │
                                                         │ FK  appointment_│
                                                         │    id           │
                                                         │    fault_desc   │
                                                         │    parts_used   │
                                                         │    total_cost   │
                                                         └─────────────────┘
```
