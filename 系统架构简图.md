## 技术栈总览

| 层级 | 技术选型 |
|:----:|:---------|
| **前端** | Vue 3 + Vite + Element Plus + Pinia + Axios |
| **后端** | Spring Boot 3.x + MyBatis-Plus + Spring Security + JWT |
| **数据库** | MySQL 8.0 |
| **文件存储** | 阿里云OSS |
| **版本控制** | Git |
