## 三、数据库表关系

```mermaid
erDiagram
    sys_user ||--o{ repair_shop : "拥有"
    sys_user ||--o{ shop_technician : "是"
    sys_user ||--o{ appointment : "创建"
    sys_user ||--o{ review : "发表"
    
    repair_shop ||--o{ shop_technician : "雇佣"
    repair_shop ||--o{ appointment : "接收"
    repair_shop ||--o{ service_type : "提供"
    
    service_type ||--o{ appointment : "关联"
    appointment ||--o{ repair_record : "生成"
    appointment ||--o{ review : "被评价"
    shop_technician ||--o{ repair_record : "完成"
    
    sys_user {
        bigint id PK
        varchar username
        varchar password
        varchar real_name
        varchar phone
        int role
        int status
        datetime create_time
    }
    
    repair_shop {
        bigint id PK
        bigint user_id FK
        varchar shop_name
        varchar address
        varchar phone
        text business_hours
        varchar avatar
        int auto_confirm
    }
    
    shop_technician {
        bigint id PK
        bigint shop_id FK
        bigint user_id FK
        varchar skill
        int status
    }
    
    appointment {
        bigint id PK
        bigint user_id FK
        bigint shop_id FK
        bigint service_id FK
        datetime appointment_time
        int status
        decimal total_price
    }
    
    service_type {
        bigint id PK
        bigint shop_id FK
        varchar name
        text description
        decimal price
        int duration
    }
    
    repair_record {
        bigint id PK
        bigint appointment_id FK
        bigint technician_id FK
        text content
        datetime complete_time
    }
    
    review {
        bigint id PK
        bigint appointment_id FK
        bigint user_id FK
        int rating
        text content
    }
```

## 四、技术栈说明

| 层级 | 技术选型 |
|------|----------|
| **前端框架** | Vue 3 + Vite |
| **UI组件库** | Element Plus |
| **状态管理** | Pinia |
| **路由** | Vue Router |
| **HTTP客户端** | Axios |
| **后端框架** | Spring Boot 3.x |
| **ORM框架** | MyBatis-Plus |
| **数据库** | MySQL 8.0 |
| **缓存** | （待扩展）Redis |
| **文件存储** | 阿里云OSS |
| **认证授权** | JWT |
| **版本控制** | Git |
